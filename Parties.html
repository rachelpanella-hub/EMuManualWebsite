Person Detail

Title

The title of the person, e.g. Mr, Ms, Dr, etc.


First

The person's first name.


Middle

The person's middle name.


Last

The person's last name (or surname/family name).


Suffix

A suffix that belongs to a person's name, e.g. Jr, III, etc.


Other Names

Multivalue field. The person's other names, e.g. Ted for Edward, Jack for John, Maggie for Margaret, etc.


Derrived Names

Salutation

The name with which the person is most usually addressed, or most commonly known, e.g. Bob for Robert, etc. Used for mail merge letters. Blank unless a name is added. Check no in the automatic field to edit the field.


Full

Automatically derived. The person's full name - title, first name, middle initial (or name) and last name. Used for mail merge letters. Check no in the automatic field to edit the field.


Brief

Automatically derived. The person's cited name - the form that the name takes in citations. Check no in the automatic field to edit the field.


Cited

Automatically derived. The person's full name - last name, first name, and middle initial (or name). Check no in the automatic field to edit the field.


Taxanomic

Automatically derived. The person's name in taxonomic reference form.. Check no in the automatic field to edit the field.
