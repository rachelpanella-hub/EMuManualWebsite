Cataloging

This is documentation for catalogers of Art, Archives and Library materials in EMu.



Modules used:



Accession Lots

This module is used to record acquisition information: values, donors, credit lines, etc. This module contains private information, most of which can only be accessed by the registration department. Accession lots allign with object files in the registration office.



Catalogue

This module is used to record information about objects: archival material, library books, art objects, crates, and frames. This is the main module staff will use and functions as a hub for other information in EMu.



Conservation

This module is used ny the Conservation department to record conservation reports and treatments for objects. This module is restricted to Registration, Curatorial and Conservation departments.



Events

This module is used to record exhbition history. The module describes the venues, dates, and objects for an event. This module is used to create checklists and to access installation photographs and ephemera.



Loans

This module is used to record loans. This module is similar to the Accession Lots module because it describes lots of objects. All loans from one lender will be recorded as one record. This module is restricted to Registration, Curatorial and Conservation departments.



Locations

This module is used to record Locations information in EMu. This module is used by Registrars to create new locations in EMu.



Parties

This module is used to record information about people. These could be artists, lenders, donors, borrowers and staff. Birth and death dates, gender, and roles are examples of some of the data captured in this module.
